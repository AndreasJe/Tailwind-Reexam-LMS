<ul>
    <li>Node.js</li>
    <li>Tailwind CSS</li>

<h2>How to use:</h2>
<p>Use this command in root to compile on-the-fly: </p>
<code>npx tailwindcss -i src/styles.css -o public/styles.css --watch</code>
